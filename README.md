# Click the Circle

A tiny static site game you can open locally. Click circles to score points before time runs out. High score is stored in localStorage.

Files

- `index.html` - game page
- `styles.css` - styles
- `script.js` - game logic

How to run

1. Open `index.html` in a browser (double-click or use a local static server).
2. Press Start or press Space to begin.
